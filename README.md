# Angular MicroFrontEnd Archicture


## Installation

Kindly follow Below commands :

#### Terminal 1:
```
cd root
npm Start..
```

#### Terminal 2:
```
cd admin
ng s –port 4201
```

#### Terminal 3:
```
cd network 
ng s –port 4202
```

####  Terminal 4:
```
cd sales 
ng s –port 4203
```



>Open http://localhost:9000/ in the browser...

That's It!!!
